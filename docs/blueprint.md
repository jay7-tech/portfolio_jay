# **App Name**: Jaydeep Portfolio

## Core Features:

- Landing Page: A visually appealing landing page that introduces Jaydeep, his skills, and experience.
- Skills Grid: Interactive grid showcasing technical skills and expertise with technologies like MERN stack, AI, and robotics.
- Innovation Showcase: Highlight key projects like YoloMart, High Five, Movifi, and Anveshana with descriptions and achievements.
- Resume Section: An up-to-date resume highlighting leadership, professional engagements, and academic achievements.
- Contact Form: Allow visitors to directly contact Jaydeep.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a modern and clean aesthetic, inspired by the original portfolio's color transitions.
- Background color: Off-white (#F5F5F5) for a clean and professional base.
- Accent color: A slightly deeper purple (#B09FDA) for interactive elements to provide a subtle contrast.
- Headline font: 'Playfair', serif for elegant, high-contrast headings.
- Body font: 'PT Sans', sans-serif paired for clean, modern body text to enhance readability.
- Use modern, minimalist icons for skills and technologies, housed in soft-rounded white 'chips'.
- Implement a floating navigation header for easy access to different sections of the portfolio.
- Incorporate scroll-triggered animations to smoothly fade elements into view.
